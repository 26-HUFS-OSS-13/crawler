# 학식 연동 AI 영양분석 앱 - Crawler

이 프로젝트는 Google Apps Script로 학교 학식 페이지에서 메뉴 정보를 수집하는 크롤러입니다.

## 주요 기능

- 학교 학식 페이지 요청
- 여러 식당의 메뉴 수집
- 날짜, 식당명, 식사 구분, 메뉴 내용 정리
- 백엔드 API로 메뉴 데이터 전송

## 먼저 읽을 문서

처음 작업하는 팀원은 아래 순서대로 문서를 읽습니다.

1. `docs/01_VIBE_CODING_GUIDE.md`
2. `docs/02_GIT_BRANCH_GUIDE.md`
3. `docs/03_CRAWLING_BASICS.md`
4. `docs/04_CRAWLER_GUIDE.md`
5. `docs/05_TASK_MEMBER_D_CRAWLER.md`

## 작업 방식

이 프로젝트는 AI 코딩 에이전트를 활용해 개발합니다.
